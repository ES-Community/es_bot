# Installation & Usage

```
npm install
npm start
```

Config settings in /config/prod.json and set the following env var:
```
DISCORD_TOKEN=
TWITTER_CONSUMER_KEY=
TWITTER_CONSUMER_SECRET=
TWITTER_ACCESS_TOKEN=
TWITTER_ACCESS_TOKEN_SECRET=
LOKI_DB_PATH=
```

## Commands available

- help
- doc
- feedtweeter
- subscribe [list]
- unsubscribe
- alert
